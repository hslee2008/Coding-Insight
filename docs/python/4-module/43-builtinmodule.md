## OS 모듈 - `system()`

### OS란?

OS는 컴퓨터의 소프트웨어 이름을 뜻합니다.

우리의 컴퓨터를 원하는 만큼 제어를 할 수 있습니다.

```py
import os
```

### OS - 파이썬 터미널 정리하기

가끔 파이썬 코드를 실행하며 많이 더러워지고 스크린에 너무 많은 정보가 쌓일 수 있습니다.

이때, 터미널에 있는 모든 내용을 다 깨끗이 없애겠습니다.

컴퓨터에 따라서 `clear` 또는 `cls` 일 수도 있습니다.
