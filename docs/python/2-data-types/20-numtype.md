## float 소수

소수란 a/b (b≠0)로 나타낼 수 있는 유리수와 a+sqrt::b의 형태로 나타낼 수 있는 무리수를 통틀어 소수라고 합니다.

조금 복잡한가요? 소수는 쉽게 소수점 뒤에 숫자가 있는 숫자를 뜻합니다.

예를 들어 3.14159는 소수점 뒤에 숫자가 있기 때문에 소수라고 합니다.
