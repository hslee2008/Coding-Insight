## 설치 되었는지 확인하기

```sh
rustc --version
cargo --version
```

다음과 같은 형식으로 뜬다면 Rust가 설치 되었습니다.

```
rustc x.y.z (abcabcabc yyyy-mm-dd)
cargo x.y.z (abcabcabc yyyy-mm-dd)
```

## 업데이트하기

```sh
rustup update
```
